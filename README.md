# Multi-Role E-Learning Dashboard

A comprehensive e-learning platform with role-based dashboards for Admin, Instructor, and Student users.

## Features

### Admin Dashboard
- User management (view, edit, delete users)
- Course overview and management
- Platform analytics and statistics
- System-wide monitoring

### Instructor Dashboard
- Course creation and management
- Student progress tracking
- Course analytics
- Student enrollment overview

### Student Dashboard
- View enrolled courses
- Track learning progress
- Browse available courses
- Access certificates

1. Open `index.html` in a web browser
2. Login with one of the demo credentials
3. Explore the role-specific dashboard
